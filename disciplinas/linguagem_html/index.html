Linguagem HTML

HyperText Markup Language (HTML) conhecida em Português como Linguagem de Marcação de Hipertexto é utilizada para composição da estrutura de uma página. Nela é possível criar títulos, parágrafos, tabelas, lista de itens, formulários, entre outros elementos visuais.

Embora possa ser utilizada sozinha. A estilização de uma página HTML geralmente acontece em conjunto com a Linguagem CSS. Através do CSS são feitos procedimentos de criação de bordas, espaçamentos, coloração, animações, etc.

Na parte funcional é comum a utilização da linguagem JavaScript. É esta que permite a validação de formulários, criação de animações complexas, desenho 2D, entre outras funções computacionais.

É através do JavaScript que é possível programar uma página Web. Observe que HTML e CSS são apenas linguagens de marcação e criação de estilos para páginas que funcionarão na World Wide Web (WWW) conhecida como Web.
