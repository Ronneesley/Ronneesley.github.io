Programação Web

A disciplina de programação Web consiste no domínio das linguagens: HTML, CSS, JavaScript. Estas são utilizadas para composição das interfaces gráficas com o usuário (GUI), também conhecida como front-end.

Nota-se que dependendo do currículo a ser formado pode haver diferentes linguagens para o funcionamento do back-end, tais como: PHP, Java, Ruby, Python entre outras.
