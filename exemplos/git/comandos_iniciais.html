Comandos iniciais para uso do Git

Como copiar um repositório?

Para copiar os arquivos de um repositório para o seu computador:

git clone URL

É sua primeira vez usando comandos Git?

Na sua primeira vez utilizando o Git será exigido a configuração do seu e-mail e do seu nome:

git config --global user.email "you@example.com"
git config --global user.name "Your Name"

Como enviar as minhas modificações?

Para adicionar a próxima submissão os arquivos que foram editados ou adicionados ao projeto:

git add -A

Para descrever o que foi realizado na submissão

git commit -m "Mensagem do que foi feito"

Para enviar os arquivos ao servidor Git, seja ele GitHub, GitLab, etc. 

git push

Como descobrir o que foi modificado?

O comando a seguir permite que você veja quais arquivos foram adicionados, removidos e modificados. 

git status

Como obter as últimas modificações?

Assim que você começar a trabalhar em um repositório clonado localmente, execute sempre o comando a seguir dentro do diretório de trabalho: 

git pull

Observações para uso do GitHub

É necessário gerar um token de acesso para submeter as suas modificações ao GitHub.
