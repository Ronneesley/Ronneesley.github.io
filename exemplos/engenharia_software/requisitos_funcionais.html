Requisitos Funcionais

Os requisitos funcionais expressam as funcionalidades do sistema. Estas funcionalidades são as ações que o usuário pode interagir com o sistema, tais como:

  • Cadastrar usuários, produtos, fornecedores, clientes, categorias
  • Emitir relatórios
  • Calcular folha de pagamento
  • Enviar e-mail de notificação

Os requisitos funcionais podem ser documentados em forma de texto ou em forma tabular.
