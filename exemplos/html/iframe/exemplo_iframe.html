O iframe permite criar uma moldura na página. Dentro desta moldura você poderá abrir outra página. Isto permite a criação de menus para navegação do conteúdo do site evitando ter que repetí-los em cada página interna.

Para isto cada link deve referir como alvo (target) aquele iframe. Perceba que o nome do iframe é dado pelo atributo name.

Recomendo que veja também a criação de layouts com tabelas. Perceba que um iframe pode ser colocado no lugar do espaço que nomeei de "conteúdo".
